Reputation System

  • Reputation Value: The amount of reputation points given to the post author when this reaction is added. Positive values increase reputation, negative values decrease it.
  • Positive: Whether this reaction is considered "positive" for display purposes.
  • Active: Only active reaction types can be used by members.
  • Display Order: Controls the order reactions appear in the picker.
